# 🎯 Site sobre Valorant  

Este projeto é um site informativo sobre o jogo **Valorant**, desenvolvido utilizando apenas **HTML e CSS**. O objetivo é apresentar detalhes sobre o jogo, personagens, mapas e mecânicas, proporcionando um layout visualmente atrativo.

## 🚀 Tecnologias Utilizadas  
- 🌐 HTML5  
- 🎨 CSS3
